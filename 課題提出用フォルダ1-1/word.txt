IT　情報技術の総称

OS
OS　コンピューターを操作するためのソフトウェア
Windows　マイクロソフト社が開発したOS
macOS (Mac OS X) アップルが開発したOS
UNIX　コンピュータ用のマルチタスク・マルチユーザーのOSの一種
Linux　オープンソースのOS

言語・環境
HTML　ウェブページを作成するための言語
CSS 　ウェブページのスタイルを指定するための言語
MySQL　データベース管理システム
PHP　動的にWebページを生成することができるサーバーサイドのスクリプト言語
JavaScript　動的なWebページを作成する事のできるプログラミング言語
jQuery　JavaScriptコードをより容易に記述できるようにするために設計されたJavaScriptライブラリ
Apache　世界的に最も普及しているWebサーバ(HTTPサーバ)ソフトウェア
Bootstrap　WebサイトやWebサービス構築の際に活用できるフレームワーク

エンジニアなど
SE　システムの設計を書くエンジニア
PG　システムプログラムのコードを書く職種
Webデザイナー　webサイトのデザインを行い、作成する人

ユーザーインターフェース
CUI　キーボードに入力されたコマンドによってコンピューターを操作する方法
GUI　コンピューターの画面上に表示されるウィンドウやアイコン、ボタン、プルダウンメニューなどを使い、マウスなどのポインティングデバイスで操作する方法

ファイルとディレクトリ
ファイル　データのまとまり
ディレクトリ　ファイルを分類、保管するための場所
ルート・ディレクトリ　層型ファイル構造における最上階層のディレクトリ
カレント・ディレクトリ　現時点で作業を行なっているディレクトリ
パス　ファイルのある場所までの経路を示すための文字列
絶対パス　目的ファイルまでの道筋を全て記述するパス
相対パス　現在位置から目的のファイルまでの道筋を記述するパス

各種プロトコル
HTTP　WebサーバとWebクライアントの間でデータの送受信を行うために用いられるプロトコル
FTP　コンピュータネットワーク上のクライアントとサーバの間でファイル転送を行うための通信プロトコル
HTTPS　HTTPを暗号通信とともにつかったもの
SSL　インターネット上でのデータの通信を暗号化し、盗聴や改ざんを防ぐ仕組み
SSH　リモートコンピュータと通信するためのプロトコル
ポート、ポート番号　コンピュータが通信に使用するプログラムを識別するための番号
HTTP、FTP、SSHの各ポート番号　80　20　22
ファイアウォール　外部ネットワークの攻撃から内部ネットワークを保護するシステム
TCP　IPと同様にインターネットにおいて標準的に利用されている プロトコル

IPアドレス
IPアドレス　ネットワーク上にある機器を識別するための数値
グローバルIPアドレス　インターネットに接続する機器が使用できるIPアドレス
プライベートIPアドレス　組織内のネットワーク（プライベートネットワーク）でのみ使用できるIPアドレス

ドメイン
ドメイン　インターネット上の住所
DNS　インターネット上で ドメイン名を管理・運用するために開発されたシステム
VPN　インターネットに接続している利用者の間に、仮想的な通信トンネルを構成したプライベートなネットワーク
ブリッジ　LANとLANを接続する方法、または接続するための中継機器
プロキシサーバー　コンピューターに代わってインターネットに接続しサイトへアクセスするサーバーを指す

電子メール
SMTP　メールを相手のメールサーバーまで届ける仕組み
POP　メールソフトが自動的にメールサーバーにアクセスしてメールをダウンロードする仕組み
IMAP　サーバにあるメールをパソコン等の端末にはダウンロードせず、サーバ上でメールを管理する仕組み

セッションとクッキー
セッション　Webサイトにアクセスして行う一連の行動
クッキー　WebサイトがスマホやPCの中に保存する情報
キャッシュ　ウェブページの情報を一時的に保存する仕組み

DB（データベース）について
チューニング　プログラムやシステム設定などを改良し、必要メモリー量や実行速度を改善すること
スロークエリ　時間のかかっているSQL(遅いSQL)のこと
インデックス　データベースで、検索や並べ替えを効率的に行うために使われる機能

テストについて
フロー　流れ
単体テスト　プログラムを構成する比較的小さな単位（ユニット）が個々の機能を正しく果たしているかどうかを検証するテスト
結合テスト　プログラムが全体として正しく動作しているかを検証するテスト

チケット管理ツール
チケット管理ツール　実施するべき作業や課題を「チケット」として管理するツール
タスク　やるべき仕事

バージョン管理
Git　プログラムソースなどの変更履歴を管理する分散型のバージョン管理システム
SourceTree　Gitを簡単に扱うためのツール
SVN　プログラムのソースコードなどを管理する集中型バージョン管理システム
コンフリクト　複数の装置やプログラムなどが同じ資源を同時に利用しようとして競合状態になってしまうこと
デグレ　修正した後のソフトウェアの品質が、修正前より悪くなることを指す
マージツール　者間および3者間のファイル比較、マージ、およびフォルダ同期を行うツール

その他
ロジック　処理の流れや手順
デバッグ　コンピュータプログラムや電気機器中のバグ・欠陥を発見および修正し、動作を仕様通りのものとするための作業
開発手法(ウォーターフォール開発、アジャイル開発)　業務システムの開発工程を構造化し、計画・制御するための枠組みのこと
アルゴリズム　問題解決のための手順や方法のこと
フローチャート図　プロセスの流れを箱と矢印を使って表現した図
設計書(外部設計、内部設計)　システムをどのように作るかという点について書かれた資料
要件定義(要求仕様書)　システムやソフトウェアの開発において、実装すべき機能や満たすべき性能などを明確にしていく作業のこと
システム開発におけるテスト(単体テスト、結合テスト、総合テスト)　プログラム中の不具合（バグ）やシステムの欠陥を発見するためにおこなう
テスト仕様書(どのようなもの、項目)　テストするポイントをまとめたドキュメント
バリデーション　対象がその仕様や文法などに照らして適切に記述・構築されているか否かを検証すること
FW（フレームワーク）　アプリケーションを開発するとき、その土台として機能させるソフトウェアのこと
PHPのFW（フレームワーク）の種類と特徴　CakePHP ,zend framework,laravel,Symfony,codeigniter,FuelPHP,Phalcon 開発時間を短くできる、多人数開発にも参加しやすくなる
Ajax　JavaScriptを使用して行われる非同期処理
API　ソフトウェアからOSの機能を利用するための仕様またはインターフェースの総称
LAMP　オープンソースソフトウェアの組み合わせを指す言葉
クラウドサーバー、レンタルサーバー、VPSの違い　サーバーを仮想化することによりクラウド上に作られたサーバー、1台のサーバーを複数の利用者で共有する、1台の物理サーバー上にユーザーごとの仮想環境を作成して提供する
AWS　Amazonが提供している100以上のクラウドコンピューティングサービスの総称
プラットフォーム　ある機器やソフトウェアを動作させるのに必要な、基盤となる装置やソフトウェア、サービス、あるいはそれらの組み合わせ（動作環境）のこと
カバレッジ　 ソフトウエア開発の最終的な検証段階における、テストが行われている範囲
エビデンス　最終検証段階においてシステムが発注どおりに動作していることを示す証拠となる書類やデータなどのこと
コンパイル　プログラミング言語で書かれた文字列（ソースコード）を、コンピュータ上で実行可能な形式（オブジェクトコード）に変換すること
バッチ処理　一定期間や一定量ごとにデータをまとめて一括して処理する方式
デプロイ(意味と方法、FTPソフトの種類)　開発したソフトウェアを実際の運用環境に配置・展開して実用に供することを指す、カナリアリリース、ブルーグリーンデプロイメント、イミュータブルデプロイメント、FileZillaやFFFTPなど
データベース接続クライアント(説明、代表的なもの)、データベース管理システム (DBMS) や関係データベース管理システム (RDBMS) 上のデータを容易に扱うためのソフトウェア、SQL Workbench/J
Gitホスティングサービス(説明、代表的なもの)　Gitによるクラウドベースのバージョン管理サービス　GitHub
チケット管理ツール(説明、代表的なもの)　実施するべき作業や課題を「チケット」として管理するツール　Trello
DBのJOINとは テーブルの結合
SEO　検索エンジン最適化
